### 1. Implement SVD

##### 목표: 데이터를 행렬 A로 받았을 때, Singular Value Decomposition을 실행해 U, S, V를 반환하는 프로그램을 작성.
###### 1.  Reduce A to upper bidiagonal matrix B using Algorithm 5.4.2.
```julia
function house(a)
  x = copy(a)
  n = length(x)
  s = x[2:n]'x[2:n]
  v = vcat(1, x[2:n])
  if s == 0.0
      b = 0.0
  else
    mu = sqrt(x[1]^2 + s)
    if x[1] <= 0.0
      v[1] = x[1] - mu
    else
      v[1] = -s/(x[1]+mu)
    end
    b = (2*(v[1]^2))/(s+(v[1]^2))
    v = v/v[1]
    end
    return v, b
end
```
